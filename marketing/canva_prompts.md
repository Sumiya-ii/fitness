## Brief 1: Instagram Feed Post Template -- Calorie Fact Card

**Purpose**: Reusable template for daily Mongolian food calorie facts
**Dimensions**: 1080 x 1080 px (Instagram square post)

**Layout**:
- Top 30%: Food photo (full width, cropped)
- Middle 50%: Dark overlay with calorie info
- Bottom 20%: CTA bar with logo

**Text Content (editable per post)**:
- Food name: "Бууз (6 ширхэг)" -- bold, 48pt
- Calorie number: "456 ккал" -- extra bold, 72pt, accent color
- Macro row: "Уураг 28г | Нүүрс ус 36г | Тос 32г" -- 24pt
- CTA: "Coach апп -- Зураг авч калори тоол" -- 20pt
- Logo: bottom right corner

**Canva Steps**:
1. Create design: Instagram Post (1080x1080)
2. Background: Set to #1A1A2E (dark navy)
3. Add photo frame at top (50% height), place food photo, apply slight darkening overlay
4. Add text block for food name -- font: Montserrat Bold, white, centered
5. Add calorie number below -- font: Montserrat ExtraBold, color #FF6B35 (orange), large
6. Add macro row -- font: Inter Regular, #CCCCCC (light gray), centered
7. Add bottom bar: rectangle, #FF6B35, full width, 15% height
8. Place logo in bottom bar, add CTA text in white
9. Save as template -- duplicate and swap food photo/numbers for each new post

**Variations to Create**:
- Khuushuur version
- Tsuivan version
- Banshtai shol version
- Cafe food version

## General Canva Tips

### Brand Kit Setup (Do This First)
1. Open Canva -> Brand Kit
2. Add brand colors: primary, accent, dark, light
3. Upload Coach logo (PNG with transparent background)
4. Set brand fonts: Montserrat (headings), Inter (body)
5. Upload phone mockup template

### Efficiency Tips
- Create one master template per brief, then duplicate for variations
- Use Canva "Brand" templates folder to organize
- Schedule Instagram posts directly from Canva Pro
- Use Canva's "Resize" feature to adapt feed posts to stories
- Canva Magic Resize can convert 1080x1080 to 1080x1920 as starting point

### Photo Sources for Food Images
- Best: Take your own photos of real Mongolian food (most authentic)
- Canva stock: Search "Mongolian food", "dumplings", "noodle soup"
- Free: Unsplash, Pexels (limited Mongolian food)
- App screenshots: Use actual Coach app screenshots for authenticity

### Export Settings
- Social media: PNG, high quality
- Print: PDF Print (300 DPI)
- Ads: PNG, keep under 20% text ratio (Meta recommendation)
